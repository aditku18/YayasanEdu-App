## 3. Pricing Rationale by Market Segment

### 3.1 Segment Analysis

#### **Segmen 1: Starter (Sekolah Kecil)**
- **Target**: TK/Playgroup, SD swasta kecil, kursus
- **Ukuran**: 50-300 siswa
- **Pain Points**: Admin manual, excel-based, biaya mahal
- **Decision Makers**: Owner/Headmaster
- **Price Sensitivity**: Tinggi
- **Value Proposition**: "Digitalisasi dengan biaya terjangkau"

| Aspek | Rationale |
|-------|-----------|
| Harga Rp 199.000 | 30% di bawah kompetitor termurah |
| Maks 300 siswa | Cukup untuk segment target |
| Fitur basic | Sesuai kebutuhan aktual |

#### **Segmen 2: Developer's Choice (Sekolah Berkembang)**
- **Target**: SD-SMP-SMA swasta menengah
- **Ukuran**: 300-800 siswa
- **Pain Points**: Rapor manual, manajemen kelas kompleks
- **Decision Makers**: Kepala Sekolah, Tata Usaha
- **Price Sensitivity**: Sedang
- **Value Proposition**: "Semua kebutuhansekolah dalam satu platform"

| Aspek | Rationale |
|-------|-----------|
| Harga Rp 399.000 | 2x Starter, tapi 50% di bawah kompetitor |
| Penilaian+Rapor | Fitur utama yang membedakan |
| Maks 800 siswa | Range ideal untuk segment |

#### **Segmen 3: Premium Solution (Sekolah Besar)**
- **Target**: SMA/SMK besar, sekolah berbasis agama
- **Ukuran**: 800-2.000 siswa
- **Pain Points**: Keuangan kompleks, parent communication
- **Decision Makers**: Kepala Sekolah, Bendahara
- **Price Sensitivity**: Rendah (ROI-focused)
- **Value Proposition**: "Solusi enterprise dengan harga fair"

| Aspek | Rationale |
|-------|-----------|
| Harga Rp 799.000 | 50% di bawah solusi enterprise lain |
| Keuangan+SPP |解决了 main pain point |
| Integrasi Pembayaran | Convenience factor |

#### **Segmen 4: Foundation Package (Yayasan)**
- **Target**: Yayasan dengan 2-20+ sekolah
- **Ukuran**: Unlimited siswa
- **Pain Points**: Tidak ada visibilitas antar sekolah, inkonsisten
- **Decision Makers**: Chairman yayasan, CEO
- **Price Sensitivity**: Sangat rendah (value-driven)
- **Value Proposition**: "Kontrol penuh atas seluruh sekolah dalam satu dashboard"

| Aspek | Rationale |
|-------|-----------|
| Range Rp 1.5-3jt | Berdasarkan jumlah sekolah |
| Multi-school | Core differentiator |
| Priority Support | Dedicated support untuk enterprise |

## 4. Upselling & Cross-Selling Strategy

### 4.1 Upselling Pathway

```
Starter (199K) 
    ↓ [6-12 bulan]
Developer's Choice (399K)
    ↓ [6-12 bulan]
Premium Solution (799K)
    ↓ [12-18 bulan]
Enterprise (1.5M+)
```

**Trigger Points untuk Upselling:**
1. **Warning Signs:**
   - Siswa >80% dari limit
   - Request fitur yang tidak tersedia
   - Penggunaan fitur advanced meningkat

2. **Automated Triggers:**
   - Notifikasi saat limit tercapai 80%
   - Popup upgrade saat akses fitur premium
   - Email upgrade suggestion bulanan

3. **Success-based:**
   - Setelah Go-Live erfolgreich
   - Setelah semester pertama
   - Saat adding new school

## 8. Billing Period Strategy

### 8.1 Billing Options

| Period | Price | Discount | Best For |
|--------|-------|----------|----------|
| **Monthly** | Rp list | 0% | New customers, uncertain needs |
| **Annual** | Rp list x 12 x 0.8 | 20% | Committed customers |
| **2-Year** | Rp list x 24 x 0.75 | 25% | Large institutions |
| **3-Year** | Rp list x 36 x 0.70 | 30% | Enterprise only |

### 8.2 Discount Structure

```
┌─────────────────────────────────────────────────────────────┐
│                   BILLING INCENTIVES                       │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  MONTHLY ─────► ANNUAL (20% OFF) ─────► 2-YEAR (25% OFF)  │
│     │                  │                        │           │
│     │    ┌────────────┘                        │           │
│     │    │                                     │           │
│     ▼    ▼         ┌──────────────────────────┘           │
│  Default        │           │                             │
│  Choice         │           ▼                             │
│               │      ┌────────────────┐                   │
│               │      │ Early Bird     │                   │
│               │      │ (Pay annual    │                   │
│               │      │ within 7 days  │                   │
```

## 11. Key Takeaways

1. **Pricing Structure**: 4-tier model covering all segments from small schools to large foundations

   - Plugin ecosystem as differentiator
   - Priority support for enterprise
   - Continuous feature improvements
